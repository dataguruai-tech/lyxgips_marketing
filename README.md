# LuxGyps Miami Artisan Hub

## Overview

This repository contains the LuxGyps Miami marketing analysis and website planning materials. The project is built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting Started

1. Install dependencies:

   ```sh
   npm install
   ```

2. Run the development server:

   ```sh
   npm run dev
   ```

3. Build for production:

   ```sh
   npm run build
   ```

4. Preview the production build locally:

   ```sh
   npm run preview
   ```

## Additional Notes

- Ensure you are using Node.js version 18 or later.
- Environment variables can be configured via a `.env` file at the project root if needed.
- Static assets such as favicons should be placed in the `public/` directory.
